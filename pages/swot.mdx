## أولا: نقاط القوة
تعتبر نقاط القوة العوامل الأساسية والداخلية الإيجابية في مؤسستك تتمثل في المميزات، هذه هي الأشياء التي تقع ضمن سيطرة مؤسستك ويمكن الاستفادة منها لتحسين استراتيجيتك

| سؤال | جواب |
| -------- | ---- |
| `ما هي نقاط قوة الشركة؟` |  خبرة اكثر من 25 عام في مجال المقاولات , جودة البناء |
| `ما الذي يحبه العملاء في علامتك التجارية؟` | التميز في اختيار المواقع والمرونة في التعامل |
| `ما هي ميزتك من حيث أصول الشركة (المالية، القوى العاملة، الإدارة، الطاقة الإنتاجية)؟` |  فريق عمل متكامل   |
| `ما هي العلاقات التجارية التي تساعدك على التنافس في السوق؟` |  المسوقين العقارين و المستثمرين   |
| `ما هي الاستراتيجيات التي عملت بشكل أفضل في الماضي؟` |  المشاركات لانشاء مشاريع شقق التمليك   |



## ثانيا: نقاط الضعف
هي العوامل الداخلية السلبية (العيوب) التي يمكن اكتشافها داخل مؤسستك ويمكن أن تؤثر على قدرتك التنافسية

| سؤال | جواب |
| -------- | ---- |
| `ما الشيء الذي يكرهه العملاء في خدماتك؟` |  لا يوجد  |
| `هل موظفيك وإدارتك مدربون بشكل كاف؟` |  نعم  |
| `ما الذي يفعله المنافسون بشكل أفضل (تقليل الأسعار، تحسين ميزات المنتج)؟` |  تحسين ميزات المنتج  |
| `هل تحتاج إلى معدات إضافية؟` | لا  |
| ` هل تصل إلى السوق المستهدف؟` |  ليس كما ينبغي |



## ثالثا: الفرص
الانفتاح في السوق أو العوامل الخارجية الإيجابية التي يمكن الاستفادة منها لتوفير فرص إضافية للنمو

| سؤال | جواب |
| -------- | ---- |
| ` هل قمت بتوسيع مجال السوق الخاص بك، أو دخلت أسواق جديدة؟ `| لا |
| ` ما هي المجالات المحتملة التي تمكنك من تطبيق نقاط قوتك؟ `| استثمار الاراضي التجارية |
| ` هل هناك أي تغييرات جديدة في الممارسات التجارية بحيث يمكن لمؤسستك الاستفادة منها؟ `| نعم |
| ` هل يعاني منافسك من مشاعر سلبية لدى العملاء، وهل يمكنك جذب هؤلاء العملاء إلى عملك؟ `| نعم |

## رابعا: التهديدات
التهديدات هي العوامل الخارجية السلبية التي يمكن أن تضر بعملك، مثل المنتجات ذات الأسعار التنافسية، أو المنتجات القادمة التي يمكن أن تؤثر على أرقام مبيعاتك

| سؤال | جواب |
| -------- | ---- |
| `هل خدمتك قادرة على تلبية طلبات العملاء؟` | نعم |
| `هل هناك أي خدمات جديدة في السوق في منافسة بشكل مباشر مع خدماتك؟` | نعم |
| `هل خدمات منافسيك أرخص وأفضل؟` | لا |
| `هل هناك تحول للعميل من منتجاتك أو خدماتك إلى منافسيك؟` | نعم |
| `هل يقوم منافسيك بتطوير خدمات أحدث وأفضل؟` | نعم |
| `هل عملك قابل للنمو ماليًا أو هل يعاني من مشاكل في التدفق النقدي؟` | قابل للنمو |
| `هل موظفيك راضون ويؤدون أفضل ما لديهم؟` | نعم |
